# Reference Library

This directory holds brewing reference books used as source material for the technique intelligence layer in `libraries/beer_research/`.

PDFs and EPUBs are gitignored — add your own copies locally. They are never committed to the repo.

## Recommended titles

The following books inform the master brewer perspective embedded in the research files. Add any you own:

| File (drop here) | Author | Covers |
|---|---|---|
| `Malt_*.pdf` | John Mallett | Malt varieties, modification, floor vs drum malting, grain character |
| `Water_*.pdf` | John Palmer & Colin Kaminski | Water chemistry, mineral targets, source brewery profiles |
| `Yeast_*.pdf` | Chris White & Jamil Zainasheff | Yeast science, pitch rate, fermentation control, strain selection |
| `For_The_Love_of_Hops_*.pdf` | Stan Hieronymus | Hop varieties, alpha acids, cohumulone, hopping techniques |
| `Brew_Like_a_Monk_*.pdf` | Stan Hieronymus | Belgian abbey brewing tradition, fermentation philosophy |

## How these are used

When asking for technique depth, recipe research, or the master brewer perspective on a style, the AI will reference knowledge drawn from these texts. The research files in `libraries/beer_research/` cite techniques and principles sourced from this library.

Add your own books. The more complete this library, the deeper the coaching.
