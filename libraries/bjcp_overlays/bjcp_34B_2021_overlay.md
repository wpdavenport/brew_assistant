# BJCP 2021 Overlay — 34B Mixed-Style Beer

Purpose:
- Constrain recipe/process choices to BJCP 2021 34B expectations.
- Highlight judge-facing boundary risks and common deductions.
- This file is NOT style doctrine. Use with the declared concept and base-style logic.

## Entry Guardrails (Critical)
- Must clearly describe what the beer is.
- Must clearly describe the base style or base-beer logic.
- Must clearly describe the specialty element, inspiration, or hybrid concept.
- Judges are evaluating both quality and whether the beer delivers the declared concept.

## Parameter Guardrails
BJCP 34B does not define universal OG/FG/IBU/SRM/ABV limits.
Use the declared concept and the beer’s internal balance.

## Sensory Targets (relative)
- Integration: High
- Concept clarity: High
- Technical flaws: Low
- Base-style recognizability: Moderate to High, depending on declaration
- Novelty: Secondary to execution

## Style Boundary Traps (judge risk)
- Beer tastes like a confused strong ale / stout / specialty beer with no clear identity
- Overbuilt formulation where sweetness, roast, and alcohol all stack up
- Declared inspiration not evident enough in the glass
- Beer is technically good but under-described on the entry form
- Process flaws mistaken for “complexity”

## Common Deductions & Prevention Focus
1) Weak concept communication
   - Be precise about base beer and inspiration
   - Tell judges what they should be perceiving

2) Poor integration
   - Keep bitterness, sweetness, roast, and specialty character in proportion
   - Avoid letting one axis dominate

3) Fermentation mismatch
   - Big mixed-style beers need a real yeast plan
   - Do not let a conceptual beer become a stuck/sweet beer

4) Fatigue in the glass
   - A memorable beer can still lose points if it is exhausting to drink
   - Preserve finish control

5) Description drift
   - If the beer is really an imperialized base style with inspiration notes, say that
   - Do not oversell clone fidelity if it is not a direct clone

## Competition-Focused Process Flags
- For 34B, entry description quality can materially change the score ceiling.
- Judges need a believable conceptual frame before they can reward execution.
- High-gravity mixed-style beers often medal when they are cleaner and more integrated than expected, not when they are bigger than expected.
