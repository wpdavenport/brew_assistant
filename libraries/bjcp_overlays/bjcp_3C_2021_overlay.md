# BJCP 2021 Overlay — 3C Czech Premium Pale Lager

Purpose:
- Constrain recipe/process choices to BJCP 2021 3C expectations.
- Highlight judge-facing boundary risks and likely deductions.
- This file is NOT style doctrine. Use with the matching 3C research file.

## Parameter Guardrails (use as constraints)
OG: 1.044-1.060  
FG: 1.013-1.017  
IBU: 30-45  
SRM: 3.5-6.0  
ABV: 4.2-5.8%

## Sensory Intensity Targets (relative)
- Malt: Medium-rich, bready, rounded, elegant
- Bitterness: Medium-high to high, pronounced but smooth
- Hop flavor: Medium to medium-high (floral/spicy noble character)
- Hop aroma: Low to medium (noble, not modern citrus/tropical)
- Ester: Low (light hop-derived fruit acceptable)
- Diacetyl: Low to none (light is acceptable, obvious is fault-risk)
- Carbonation: Medium-high (style range 2.3-2.7 vols)

## Style Boundary Traps (judge risk)
- Harsh bitterness or minerality instead of rounded bitterness
- Thin, one-dimensional malt body (reads generic pils instead of Czech depth)
- German-leaning crisp-dry profile without Czech malt fullness
- Excess sulfur, green lager character, or yeast bite
- Obvious diacetyl (butter/butterscotch) beyond subtle tolerance
- Modern hop character (citrus/tropical/dank) overtaking noble profile
- Oxidation dulling noble hop expression and finish

## Common Deductions & Prevention Focus
1) Rough bitterness
   - Keep sulfate moderate and chloride balanced for rounded bitterness.
   - Favor noble hops and smooth boil/whirlpool integration.

2) Underdeveloped malt complexity
   - Build layered bready malt character, not just generic base malt flavor.
   - Ferment cleanly to preserve malt/hop balance.

3) Fermentation flaws (diacetyl/sulfur/green beer)
   - Use disciplined lager pitch rates and temperature control.
   - Gate packaging on stable FG and clean forced VDK.

4) Out-of-style hop profile
   - Keep hop choices and aroma character noble/classic.
   - Avoid IPA-like late hop signatures.

5) Conditioning faults
   - Allow sufficient cold maturation before judging.
   - Avoid rushing to package while flavor is still angular.

## Competition-Focused Process Flags
- If bitterness tastes sharp instead of firm-rounded, water/hop balance is off.
- If beer is very dry and snappy but lacks malt depth, it may score as more German than Czech.
- If diacetyl is questionably present warm, wait and re-test before crash/package.
- Enter only when clarity is brilliant and flavors are integrated after lagering.
