# BJCP Study Rubrics

## Scoring Bands
- `90-100`: exam ready
- `80-89`: near ready, patch weak tags
- `70-79`: foundational gaps remain
- `<70`: rebuild fundamentals before timed mocks

## Adaptive Quiz Rules
- Correct answers increase difficulty within the same tag.
- Incorrect answers trigger one immediate follow-up question in that tag.
- If two misses occur in same tag, emit a short remediation block before continuing.

## Feedback Format
For every missed question:
1. Correct answer
2. Why it is correct
3. Why common distractors are wrong
4. One practical memory cue

## Readiness Report
Include:
- Total score
- Score by tag
- Top three weak tags
- Recommended next drill set

## Exit Criteria for "Mock Priority"
- At least 50 practice questions answered
- Off-flavor and style-comparison tags both >= 80%
- Last two mixed quizzes >= 80%
