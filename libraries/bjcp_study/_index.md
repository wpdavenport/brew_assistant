# BJCP Study Index

This folder contains study assets for BJCP exam prep mode.

## Files
- `curriculum.md`: topic sequence and lesson goals.
- `rubrics.md`: grading rules, scoring bands, and readiness criteria.
- `question_bank.json`: tagged practice questions.
- `progress_template.json`: session/progress data shape.

## Mode Contract
- Study mode is opt-in only.
- Enter with: `enter bjcp mode`
- Exit with: `exit bjcp mode`
