# Batch Logs

Store completed brew reports here, not in `recipes/`.

## Naming convention
- `YYYY-MM-DD_style_slug.md` for written logs
- `YYYY-MM-DD_style_slug.html` for printable/export logs

## Examples
- `2026-02-20_copper-crown-esb.md`
- `2026-02-20_copper-crown-esb.html`

## What belongs where
- `recipes/`: recipe specs and planning artifacts
- `batch_logs/`: executed brew reports and post-brew markdown logs
- `brewing/brew_day_sheets/`: printable brew-day sheets and tips pages

## Brew Log Standard
- Use `batch_logs/brew_log_template.html` as the canonical printable log baseline.
- When creating a new recipe-specific printable log, copy the template and fill placeholders; do not start from scratch.
